## Projects

### ЁЯеЛ Georgian Wushu Competition System
> *Full-stack platform тАФ deployed and used in production by the Georgian Wushu Federation*

I built this because the problem was real: Georgian Wushu tournaments had no digital infrastructure. Everything from athlete registration to bracket generation to live scoring was manual.

The platform handles **athlete registration and seeding**, **automated bracket generation**, **real-time match scoring via WebSockets**, **JWT-based role access** for admins and referees, and **responsive admin dashboards** for live tournament operations.

`NestJS 11` `Next.js 16` `PostgreSQL` `Prisma` `Socket.IO` `JWT` `Tailwind CSS` `Docker`

[![Repo](https://img.shields.io/badge/Repository-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/DavitNazarov/Wushu)
