## Profile

I am an 18-year-old developer from Georgia focused on shipping production-oriented web applications, not tutorial projects.

My work includes a **competition management platform actively used by the Georgian Wushu Federation**, a **dynamic portfolio with protected CMS**, and a **social movie discovery platform with AI-powered recommendations**.

Currently working as a **Junior Front-End Developer at SoftGen Group** and preparing to start **BSc IT & Network Engineering at FH Kärnten, Austria** in September 2026.

Outside of software, I am a **European Wushu Champion** — 19th European Championships, 60 kg Light Sanda, Gold, 2024. That background shaped the way I work: disciplined, direct, consistent, and outcome-focused.

## Working Principles

```txt
Requirements first.
User flows before components.
Backend integration before polish.
Edge cases before launch.
AI assistance, manual ownership.
```

I break work into screens, states, permissions, data dependencies, and failure cases.  
I prefer to connect the interface to the backend early so the product behaves like a real application from the first useful version.

> Assisted by AI. Reviewed manually. Responsibility stays with me.
